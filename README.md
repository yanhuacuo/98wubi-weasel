## 使用说明

**给任意安装的「小狼毫」添加完整的98五笔配置**

小狼毫上的「98五笔」配置文件，已经非常完善。已从其它渠道安装过小狼毫的朋友，可以利用本档迅速获取完整的98五笔配置。

1：以管理员权限运行「部署98五笔.bat」,在任意「小狼毫」上部署98五笔。

2：安装「98WB-0.otf」矢量字体。

3：98超集字体支持，请到「98五笔资源库」的「字体支持」处完成。
